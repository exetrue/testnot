# NOTLAAN Güvenli Not Defteri

Flutter ile hazırlanmış PIN korumalı not, hatırlatma ve şifre kasası uygulaması.

## APK alma

```bat
cd E:\notlaan_flutter_ready
flutter pub get
flutter build apk --release
```

APK:
`build/app/outputs/flutter-apk/app-release.apk`

## GitHub Actions
Projeyi GitHub'a yükleyin. Actions > Build Flutter APK çalışınca APK Artifacts bölümüne düşer.
